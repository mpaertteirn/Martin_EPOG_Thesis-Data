# WTO Transition Impacts on the Accumulation-Metabolism Nexus of the United States -- Replication kit

This repository contains the R code and data to replicate the results and figures in Peter Martin's EPOG-JM 2024-2026 master's thesis, "WTO Transition Impacts on the Accumulation-Metabolism Nexus of the United States."

---

Make sure to paste a FRED API Key into "FRED_API.txt" and unzip the "ameco0_csv.zip" file for the code to run properly.

The "Online_data_USA.xlsx" is the original data from Gierlinger & Krausmann (2012), which was rearranged into a more compatible formatting in the "Online_data_USA_material.csv" file. 

Make sure your R version and the packages are up-to-date. This code was tested under R 4.5.2 (2025-10-31).

## Results

The code generates the results and figures of the main thesis and its supplementary appendix.
